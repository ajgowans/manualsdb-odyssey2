# The MiSTer Manuals DB - Magnavox Odyssey 2

This is a database for the MiSTer project that downloads the English manuals in .pdf form for the Magnavox Odyssey 2 to the docs folder, which can be loaded from the OSD within th core by selecting "Help".

To use it simply copy and paste the below to the bottom of your downloader.ini file (found at: /media/fat/downloader.ini )

Or download this .ini file and put it in /media/fat https://github.com/ajgowans/manualsdb-odyssey2/blob/db/downloader_ajgowans_manualsdb-odyssey2.ini


```ini
[ajgowans/manualsdb-odyssey2]
db_url = https://raw.githubusercontent.com/ajgowans/manualsdb-odyssey2/db/db.json.zip
